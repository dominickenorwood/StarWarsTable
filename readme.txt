STAR WARS TABLE.

1. Developed for Chrome but should work for FireFox and Safari.

2. This table reaches out for an endpoint at https://swapi.co/api/species/1/
   and returns a list oh humans in the Star Wars universe.

3. Humans are sorted alphabetically by last name.

4. Each table row displays the human's name, height (written in meters,
   i.e. 1.77m), mass (written in kilograms, i.e. 76kg), and hair color.

5. The dashboard sums up the following bullets :
     - The average mass of all humans in the table.
     - The name and height of the tallest human.
     - The most common hair color of all humans.

6. Open index.html locally
